    Passenger Real Time Bus Route Display

  • Built with Sencha ExtJS4\Touch using Sencha Architect + Google Maps V3 API

    • A live updating Google Map display, showing a bus route overlay and the current locations of any vehicles operating on the route.

    View Demo 

    Notes:

  • If you don't see any vehicles operating on the route displayed on the map, then please check this timetable and view the demo again when the next scheduled service is due to take place.
  • The bus route overlay displayed on the map is provided from a KML file, loaded through the Google Maps API.
  • When viewing the demo on a mobile device you will be redirected to a Sencha Touch version of the application (source is minified).
  • The source code for the desktop version is not minified so feel free to view the source on the page for a closer look at what's going on under the hood.
  • This application makes use of token based authentication through a Url query string parameter passed in each request made to the server.
