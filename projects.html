Pinpointers Web Application

  • ExtJS 3
  • Google Maps API v3
  • AJAX/WebSocket
  • SQL Reporting Services

The primary web application used by customers for locating vehicles in their fleet fitted with Pinpointers tracking devices.

Users are able to view current and historial position data, including speed and heading. The application makes use of AJAX requests, and also displays information on a Google Maps display.

  • Sencha Touch 2
  • Google Maps API v3
  • AJAX

A companion mobile version of the main web application, wrapped into a native iOS and Android app using PhoneGap Build, distributed on Google Play and the AppStore.
